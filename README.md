# Akumi Python SDK

Official Python client for the Akumi EU-sovereign inference API. Built on
httpx, with both a synchronous and an asynchronous client. Python 3.9+.

Synchronous:

    from akumi import Akumi

    client = Akumi.from_api_key("sk-...")
    result = client.chat.create(params={"model": "akumi-eu-1", "messages": []})

Asynchronous:

    import asyncio
    from akumi import AsyncAkumi

    async def main():
        client = AsyncAkumi.from_api_key("sk-...")
        result = await client.chat.create(params={"model": "akumi-eu-1", "messages": []})
        async for chunk in client.chat.create_streamed(params={"model": "akumi-eu-1", "messages": [], "stream": True}):
            print(chunk)

    asyncio.run(main())
